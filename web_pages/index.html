A.D.E.G.A

O A.D.E.G.A (Ánalise de Dados Estatísticos da Grade Acadêmica) é um sistema web feito, a priori, para auxiliar os coordenadores de cursos a administrarem a graduação. O sistema oferece análises como o IRA médio do curso, listagem de possívei formandos, a taxa de reprovação das disciplinas ou a quantidade de evasão por semestre. Essas análises buscam auxiliar os coordenadores a identificar problemas na grade do curso e monitorar melhor o desempenho de alunos.

Manual de instalação

Para instalar o projeto devemos clonar o seu repositório no GitHub. Para isso execute este comando em seu terminal:

$ git clone git@gitlab.c3sl.ufpr.br:adega/adega.git

Instalando o Docker

O projeto foi desenvolvido usando Docker, um software contêiner que facilita a virtualização, desenvolvimento e manutenção de aplicações. Siga com os comandos a seguir para executar o ADEGA com o docker (Docker >= 1.13.1 e Docker-compose >=1.21.2)

$ git clone git@gitlab.c3sl.ufpr.br:adega/adega.git # Comando já executado
$ cd adega/
$ sudo make docker-install # Instala o docker.io e docker-compose
$ sudo apt install docker.io
$ sudo make docker-up # Executa os containers postgres e webserver

Pronto. A partir deste momento o projeto esta rodando e pode ser acessado em localhost:8000
