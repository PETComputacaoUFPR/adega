A.D.E.G.A.

O A.D.E.G.A (Ánalise de Dados Estatísticos da Grade Acadêmica) é um sistema web feito, a priori, para auxiliar os coordenadores de cursos a administrarem a graduação. O sistema oferece análises como o IRA médio do curso, a taxa de reprovação das disciplinas ou a quantidade de evasão por semestre. Essas análises buscam auxiliar os coordenadores a identificar problemas na grade do curso e monitorar melhor o desempenho de alunos.
